As a StickyNotes user, I want the auto theme option to automatically switch between light and dark modes based on my system settings so that the app seamlessly adapts to my operating system's appearance without manual intervention.

**US-003: Theme Persistence**
As a StickyNotes user, I want my selected theme to be remembered across app sessions so that I don't have to reconfigure my visual preferences every time I open the application.

**US-007: Font Family Selection**
As a StickyNotes user, I want to choose from a selection of fonts (serif, sans-serif, monospace, handwriting) so that I can improve readability and match my personal style preferences.

**US-009: Font Weight Options**
As a StickyNotes user, I want to select different font weights (light, regular, medium, bold) so that I can emphasize important content and create visual hierarchy in my notes.

**US-010: Note Shape Variants**
As a StickyNotes user, I want to choose from different note shapes (square, rectangle, rounded corners, circle) so that I can create visual variety and organize different types of content with distinct appearances.

**US-011: Note Shadow and Border Effects**
As a StickyNotes user, I want to apply shadow effects and border styles to my notes so that I can create depth and visual separation that makes my workspace more organized and aesthetically pleasing.

**US-012: Note Size Presets**
As a StickyNotes user, I want to select from predefined note sizes (small, medium, large, custom) so that I can quickly create notes appropriate for different amounts of content without manual resizing.

**US-013: Transparency Options**
As a StickyNotes user, I want to adjust the transparency/opacity of my notes so that I can create layered visual effects and see underlying content when needed.

## Layout Preferences

**US-014: Grid and Alignment Options**
As an organized user, I want grid snap and alignment options so that I can arrange my notes in clean, structured layouts that look professional and are easy to navigate.

**US-015: Workspace Background Customization**
As a StickyNotes user, I want to customize my workspace background (solid colors, gradients, patterns, images) so that I can create a personalized environment that inspires creativity and productivity.

**US-016: Note Spacing Controls**
As a StickyNotes user, I want to control the default spacing between notes and margins so that I can optimize the use of screen real estate according to my workflow preferences.

**US-017: Theme Export**
As a StickyNotes user, I want to export my custom themes to files so that I can back up my visual configurations and share them with colleagues or across devices.

**US-018: Theme Import**
As a StickyNotes user, I want to import theme files created by myself or others so that I can quickly apply new visual styles and benefit from community-created themes.

**US-019: Theme Sharing**
As a creative user, I want to share my custom themes with other StickyNotes users through a built-in sharing mechanism so that I can contribute to the community and discover themes created by others.

**US-020: Real-time Theme Preview**
As a StickyNotes user, I want to see a real-time preview of how themes will look in my workspace before applying them so that I can make informed decisions without disrupting my current work.

**US-021: Theme Gallery with Previews**
As a StickyNotes user, I want to browse a gallery of available themes with visual previews so that I can quickly identify and select themes that appeal to me without having to test each one individually.

**US-022: Preview with Sample Content**
